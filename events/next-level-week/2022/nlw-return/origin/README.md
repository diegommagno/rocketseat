<h3 id="concepts-and-skills">ğŸ“š Concepts and Skills</h3>

- Semantic HTML structure for a marketing landing page  

- SCSS organization using variables, base styles, and layout rules  

- Responsive layout composition with Flexbox and CSS Grid  

- Header style changes based on scroll position using JavaScript  

- Mobile menu open and close state controlled via CSS classes  

- Basic DOM manipulation with `querySelector` and `classList`  

- Separation of concerns between structure, styling, and behavior  

- Initial UI scaffold intended as a foundation for further development
