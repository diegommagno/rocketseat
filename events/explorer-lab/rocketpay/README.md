<h3 id="concepts-and-skills">ğŸ“š Concepts and Skills</h3>
<hr style="opacity: 0.3;" />

- Semantic HTML structure for form fields and card presentation  

- Responsive layout composition using CSS Grid and Flexbox  

- Scalable typography and spacing using `rem` units  

- Modular CSS organization with separated base, form, and component styles  

- Real-time input masking using **iMask.js** for credit card number, expiration date, and security code  

- Dynamic mask dispatch logic based on card number patterns (Visa, Mastercard, default)  

- DOM manipulation with `querySelector`, `innerText`, and `setAttribute`  

- Visual card updates driven by user input events  

- JavaScript event handling with `addEventListener` for form interaction  

- UI state updates based on validated masked input  

- Prevention of default form submission behavior for controlled interactions  

- Asset and module bundling using **Vite** with ES modules  

- Multilingual structure with separate implementations for PT-BR and EN  

<br/>

<p align="center">
This project demonstrates how to combine form validation, input masking, and dynamic UI updates to simulate a real-world credit card interaction experience using modern frontend tooling.
