<h3 id="concepts-and-skills">üìö Concepts and Skills</h3>
<hr style="opacity: 0.3;" />

- Use of CSS custom properties (variables) to define theme colors  

- Theme switching through class changes on the root HTML element  

- Attribute selectors with `input[type="radio"]` and checked states  

- Custom radio button styling using `appearance: none`  

- Fixed positioning and horizontal centering using `left: 50%` and `transform: translateX(-50%)`  

- Basic DOM selection with `querySelectorAll`  

- Event handling using click listeners  

- Separation of visual themes (light, dark, blue, green) through CSS classes  

<br/>

<p align="center">
This project focuses on practicing CSS variables and basic JavaScript DOM manipulation to implement a lightweight theme switching mechanism.
