<h3 id="concepts-and-skills">ğŸ“š Concepts and Skills</h3>

- Semantic HTML structure to clearly separate product image and product information  

- Global CSS reset and consistent box sizing using `* { box-sizing: border-box }`  

- Use of CSS custom properties for theme colors and centralized styling control  

- Normalized root font-size (`62.5%`) to simplify `rem`-based spacing and typography  

- Layout composition using Flexbox for centering and alignment  

- Absolute positioning for interactive UI controls within the product container  

- Visibility control through utility classes (`.hidden`) using `opacity` and `visibility`  

- Hover states and transitions for interactive elements to improve user feedback  

- JavaScript DOM manipulation with `querySelector` and `addEventListener`  

- Dynamic UI state switching via class toggling and image source replacement
