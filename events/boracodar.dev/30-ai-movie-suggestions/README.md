<h3 id="concepts-and-skills">ğŸ“š Concepts and Skills</h3>
<hr style="opacity: 0.3;" />

- Semantic HTML structure for dynamic content rendering  

- Asynchronous JavaScript using `async` / `await` with `fetch` API  

- Integration with an external REST API to retrieve movie data  

- Data transformation and formatting from API responses  

- Randomized selection logic using `Set` to avoid duplicate recommendations  

- Dynamic DOM rendering based on API results  

- Event-driven interaction to trigger trailer playback  

- Asset organization using separate `icons` and `img` directories  

- Scalable spacing and typography using `rem` units
