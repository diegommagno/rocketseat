<h3 id="concepts-and-skills">ğŸ“š Concepts and Skills</h3>
<hr style="opacity: 0.3;" />

- Semantic HTML structure separating each form step into independent sections  

- Progressive disclosure pattern to guide users through multi-step workflows  

- Layout composition using Flexbox and CSS Grid for consistent form alignment  

- Scalable spacing and typography using `rem` units  

- UI state control via utility classes (`.hidden`) to manage step visibility  

- JavaScript-driven navigation between steps using `addEventListener`  

- Forward and backward navigation logic to improve user control  

- Delayed visual feedback on form completion using `setTimeout`  

- Integration of the `js-confetti` library to provide success feedback
